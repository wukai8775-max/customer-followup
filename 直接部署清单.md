# 直接部署清单

## 需要上传到 GitHub 的内容

上传当前项目根目录的全部文件，不要只上传 `dist`。

必须包含：

```text
src/
netlify/functions/
supabase/schema.sql
package.json
netlify.toml
tsconfig.json
vite.config.ts
index.html
```

## Supabase

- 运行 `supabase/schema.sql`
- 不需要创建管理员
- 不需要创建销售账号
- 不需要开启公开匿名表访问

## Netlify 环境变量

```text
SUPABASE_URL
SUPABASE_SECRET_KEY
APP_ACCESS_PASSWORD
SESSION_SECRET
```

## 不要上传

```text
.env
node_modules/
dist/
.netlify/
```
